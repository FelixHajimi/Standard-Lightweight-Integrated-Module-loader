# 本文将讲述操作员模式的所有命令及其如何使用

## 如何运行操作员模式
```bash
# 你一般使用SLIM是这样的
slim script subscript args...
# 但是要使用操作员模式需要在前面加上--admin标识
slim --admin adminscript args...
```

## help
### 示例
```bash
slim --admin help
slim --admin help fex
```
### 参数格式
```bash
slim --admin help [id]
```
## 解析
其中,如果没有id参数,则直接输出所有命令的ID和它的参数格式  
例如`slim --admin help`
```txt
file : -
file.add : - <path>
file.del : - <path>
file.load : - <path> [encoding:utf-8]
file.move : - <path> <newPath>
file.rename : - <path> <newName>
query : -
query.tree : - <mode> [data] [configs]
query.path : - [path]
query.cmd : - [id]
fex : - <path> [encoding:utf-8] [plugin]
```
如果含有id参数,则输出该命令的ID和参数格式
例如`slim --admin help fex`
```txt
fex : - <path> [encoding:utf-8] [plugin]
```

## create
### 示例
```bash
slim --admin create
slim --admin create test
slim --admin create test "- <name>"
```
### 参数格式
```bash
slim --admin create [id] [format]
```
### 解析
如果`id参数`和`format参数`都没有写,则会扫描`command_config`然后判断命令ID是否在`command_dir`  
如果不在,则在`command_dir`创建并写好参数

如果写了`id参数`,则会在`command_config`和`commadnDir`新增这个`id`,但是参数格式是`-`

如果写了`format参数`和`id参数`,则会在`command_config`和`commadnDir`新增这个`id`和对应的`format`
