# 本文将讲述config函数API的所有tools的用法

## tran-Tran
这是一个翻译工具,见程序第180行
内置有一个`run`方法,可以返回一个字符串作为翻译文本
### 示例
```py
tran = None
TRAN = {
    "zh-cn": {
        "hello": "你好!"
    },
    "en-us": {
        "hello": "Hello!"
    },
}
def config(tools: dict[str, any], lang: str, **args):
    global tran
    tran = tools["tran"](TRAN, lang)

def enter(name: str, **args):
    print(tran.run("hello")+name)
```
运行`slim hello Felix`且language为`en-us`时会输出"Hello!Felix"  
`run`函数有一个优先级:  
先尝试在`TRAN`中使用`lang`  
=> 如果不存在则尝试使用`en-us`  
=> 如果还是不存在则使用第一个语言  
=> 否则报错

`run`函数还有一个参数:`content`,他表示如何输出这个字符串,这个参数默认是"\<?\>"  
在传入`content`时,"\<?\>"标识符表示翻译后输出的文本  
像上面的例子就可以用这种方法
```py
# enter/
print(tran.run("hello", f"<?>{name}"))
```

如果要面对复杂场景则可以使用eval来输出,还是上面的例子
```py
# TRAN/en-us/
{"hello": "f\"Hello!{name}\""}

# enter/
print(eval(tran.run("hello")))
```
