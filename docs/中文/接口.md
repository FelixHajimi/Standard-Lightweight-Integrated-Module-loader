# 本文将讲述命令的所有API接口

## 示例
```py
# script.py
def config(path: str, lang: str, debug: bool, tools: dict[str, any]):
    pass

def enter(**args): # 这里是你在配置里面写的所有参数配置
    # 具体业务逻辑
    pass
```

### config函数的作用
它表示程序全局的设置,程序将以参数形式将全局设置传入此函数  
参数详解
- path,主程序的路径
- lang,全局语言,可配合`tools["tran"]`使用
- debug,全局调试模式,可以用调试模式输出详细信息
- other,包含的其他设置
- tools,可能会用到的工具,目前包含Tran一个工具,见程序第180行或查看"工具"文档
