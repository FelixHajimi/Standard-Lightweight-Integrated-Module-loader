# 本文将讲述`setting.json`如何配置
> 为什么要有`setting.json`?  
> `setting.json`存储了所有的全局设置,没有它程序将不会运行
## 示例
```json
{
  "language": "en-us",
  "command_config": "command.json",
  "command_dir": "command",
  "debug": false,
  "other": {...}
}
```

- language为全局语言,比如en-us,表示所有命令统一用英语输出/记录日志
- command_config为命令配置,command.json就表示所有命令配置都存储在command.json文件中
- command_dir为命令调用文件夹,在读取到了command_config之后调用文件会在这个文件夹里面调用
- debug为调试开关,命令可以通过debug来进行调试
- other为其他设置,命令可以调用这个键获取设置
